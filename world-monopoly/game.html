📋 Aturan World Monopoly

Tujuan: Kuasai sebanyak mungkin negara dan buat pemain lain bangkrut!

Giliran: Lempar dadu → Maju sesuai angka → Lakukan aksi di petak.

Beli Negara: Jika petak kosong, kamu bisa membelinya.

Bayar Denda: Jika petak milik pemain lain, kamu harus bayar denda.

Bangun Rumah: Berhenti di negaramu sendiri untuk membangun rumah (menaikkan denda).

Penjara: Masuk penjara jika kena petak "Go to Jail". Butuh double dadu atau kartu untuk bebas.

START: Setiap melewati START mendapat bonus uang.

Bangkrut: Jika uang habis, kamu bangkrut dan keluar dari permainan.
